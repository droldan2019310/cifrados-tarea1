# Ejercicio: Criptografía  
Historia – Conversiones – Cifrado con Llaves

---

## Descripción general

Este proyecto contiene la resolución completa del ejercicio de criptografía, dividido en tres partes principales:

1. Investigación histórica de un cifrado  
2. Conversión entre ASCII, Binario y Base64  
3. Generación de llaves dinámicas y creación de un nuevo cifrado  

El objetivo es comprender la evolución de la criptografía, el manejo de representaciones de datos y la creación de un cifrado propio utilizando principios básicos como binario, ASCII y XOR.

---

## Requisitos

- Python 3.8 o superior  
- No se utilizan librerías externas  
- Sistema operativo Windows, Linux o macOS
